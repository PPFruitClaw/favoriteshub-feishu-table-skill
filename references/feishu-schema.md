# 飞书多维表格字段规范

## 子表命名

- `github`
- `x`
- `xiaohongshu`
- `douyin`
- `other`

## 字段清单（所有子表一致）

- `所属平台`：文本（建议后续改成单选）
- `链接`：URL 或文本
- `内容梗概`：多行文本
- `收藏或星标数量`：数字（允许为空）
- `收录时间`：日期时间

## 初始化建议

1. 新建多维表格应用（app）。
2. 在默认表或目标子表中创建上述字段。
3. 为每个平台准备同名子表，字段结构保持一致。
4. 首次写入前用 `所属平台 + 链接` 做去重检查，避免重复记录。

## 写入映射

- `platform` -> `所属平台`
- `link` -> `链接`
- `summary` -> `内容梗概`
- `favorite_or_star_count` -> `收藏或星标数量`
- `ingested_at` -> `收录时间`
